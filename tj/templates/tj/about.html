About this Site

Purpose

This website provides a query interface to the OECD's Creditor Reporting System (CRS) data, much like OECD's QWIDS site.

The intention is to build a repository of international aid data related to Transitional Justice (TJ) as recorded in the CRS database. TJ data is not clearly designated as such in CRS, so manual classification has been performed on the results of ad-hoc text queries in various languages.

You can review the results of this classification here.

Methodology

Raw CRS data is downloaded from the OECD and filtered on various criteria:

  • "Purpose Codes" that might potentially refer to TJ work
  • Aid data from the year 2000 onwards
  • Columns relevant to the project
This reduced dataset is loaded into a database along with custom classification fields. This website is then used to query and classify the unanalyzed data and review the analyzed results.

The data processing and website code is open source, and available on github.
