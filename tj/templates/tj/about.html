About this Site

Purpose

This website provides an interface to search the OECD's Creditor Reporting System (CRS) data, much like OECD's QWIDS site.

The intention is to build a repository of international aid data related to Transitional Justice (TJ) as recorded in the CRS database, which includes all bilateral assistance from OECD member states and some multilateral assistance. TJ data is not clearly designated as such in CRS, so manual classification has been performed on the results of ad-hoc text queries in various languages (e.g. "transitional justice", "truth", "ICTY", etc).

You can review the results of this classification here.

Methodology

Raw CRS data was downloaded from the OECD on 2014-01-30 and filtered on various criteria:

  • "Purpose Codes" that might potentially refer to TJ work
  • Aid data from the year 2000 onwards
  • Columns relevant to the project
This reduced dataset was loaded into a database along with custom classification fields. This website was then used to query and classify the unanalyzed data and review the analyzed results.

The data processing and website code is open source, and available on github.

People

Paige Arthur is the primary persion responsible for querying and coding the grant data. She is Principal of Public Action Research. You can contact her at parthur@publicactionresearch.com.

Andrew Rosenfeld wrote the code to collect and transform OECD data and interact with it via this web site.

The development of the database is supported through a grant from the Bureau of Democracy, Human Rights, and Labor of the U.S. State Department, as well as through pro bono work from Andrew Rosenfeld.
